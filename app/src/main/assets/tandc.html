Permissions and Privacy Policy for Focus Launcher

Last Updated: [Insert Date]

Focus Launcher helps users improve productivity by temporarily restricting access to selected apps. Users may unlock apps early by paying a fee.

Permissions Required

1. Accessibility Service

Focus Launcher uses the Android Accessibility Service only to:

  • Detect when a restricted app is launched.
  • Enforce app locking during the restricted period.
  • Display focus reminders (without notifications).

Note: We do not collect, log, or transmit any sensitive user data accessed through this service.

2. Query All Packages (QUERY_ALL_PACKAGES)

This permission allows the app to:

  • Display a list of installed apps so users can select which apps to restrict.
  • Verify if a restricted app is launched.

Premium Unlock Feature

Users may unlock restricted apps before the scheduled time by paying a fee. All payments are processed securely via Google Play Billing.

Privacy Policy

Data Collection

Focus Launcher does not collect or store:

  • Personal user data (emails, contacts, etc.).
  • App usage history beyond the local device.
  • Accessibility service data (keystrokes, text inputs, etc.).

Data Processing

  • Local-Only Processing: All app restrictions and timers are managed locally on the device.
  • No Backups: We do not upload or sync data to external servers.

User Control

Users can:

  • Revoke permissions anytime via device settings.
  • Uninstall the app to remove all local data.
