Permissions Overview

Accessibility Service

This service is used solely to:

  • Monitor launch of selected restricted apps
  • Temporarily limit access during focus periods
  • Show helpful reminders to support focus goals
Note: No personal or sensitive user data is accessed, collected, or transmitted through this service.

App Usage Access

This permission helps Focus Launcher:

  • Display a list of installed apps for restriction setup
  • Check if restricted apps are in use (locally only)

Premium Features

Users can unlock access to restricted apps before the scheduled time by making a secure payment via Google Play Billing. No financial data is handled by Focus Launcher directly.

Privacy Policy

Data Collection

We do not collect or store the following:

  • Email addresses, contacts, or other personal identifiers
  • App usage data beyond the device
  • Accessibility data such as keystrokes or text input

Data Processing

  • On-Device Only: All operations run locally on your device
  • No External Sync: Your data is never uploaded or shared

User Controls

  • You can revoke permissions at any time via system settings
  • Uninstalling the app removes all locally stored data

Compliance

  • Compliant with Google Play’s Content Policies
  • Supports GDPR and CCPA standards where applicable

Updates to This Policy

Any future updates will be made available both within the app and on this page. Your continued use indicates acceptance of the updated terms.
