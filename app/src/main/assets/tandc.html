Permissions and Privacy Policy for Addiction App

Permissions Required for Addiction Control Features

To provide addiction control services and help users manage app usage, Addiction App requires the following permissions:

1. Foreground Services and Notifications

We use the following permissions to run essential services:

  • FOREGROUND_SERVICE: This permission enables our app to run foreground services that monitor app usage in real-time.
  • FOREGROUND_SERVICE_DATA_SYNC: This permission allows us to sync data necessary for displaying reminders and timers, helping users stay aware of restricted app usage.

2. Usage Access (PACKAGE_USAGE_STATS)

PACKAGE_USAGE_STATS: This permission allows us to monitor which apps are currently being used. This functionality is critical for alerting users when a restricted app is opened. Users must grant this access in system settings for the app to function as intended.

3. All Apps Query (QUERY_ALL_PACKAGES)

QUERY_ALL_PACKAGES: We request this permission to display a list of all installed apps on your device. This feature allows users to select which apps to restrict, enabling a personalized list of restricted applications. This permission is essential for the appâ€™s core functionality.

4. Notification Access (POST_NOTIFICATIONS)

POST_NOTIFICATIONS: This permission is used to send notifications to users when restricted apps are in use and to show usage time. Users can manage these notifications within the app settings.

Privacy and Data Security

Commitment to User Privacy

We value your privacy. All data related to app usage is processed locally on your device and is not shared with external parties. We do not collect, store, or transmit any personally identifiable information (PII) without your explicit consent.

Data Processing

  • Local Data Processing: All app usage data is processed locally to ensure that your information remains confidential. We do not upload or back up any data to external servers.
  • User Control: Users have full control over their data and can revoke permissions at any time through the app settings or device settings.

Data Retention

We retain data only for as long as necessary to fulfill the purposes outlined in this policy. Once it is no longer needed, we securely delete any data stored locally on your device.

Compliance with Regulations

Addiction App is committed to complying with all relevant data protection regulations, including the General Data Protection Regulation (GDPR) and the California Consumer Privacy Act (CCPA).

Changes to This Policy

We may update this permissions and privacy policy from time to time. Any changes will be posted within the app, and users will be notified. We encourage users to review this policy periodically for the latest information on our practices.
